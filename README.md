### 今後の展望
#### 現在の問題点
- ブラウザで動くWebアプリなのでリアルタイム性が低い
- 投げ銭はアプリ内のポイントでしかないため、ユーザーにとっての価値が低い

#### ではどうしていきたいか
- 最終的にはブロックチェーンの技術を用いてGPSで位置情報を取ることなくサーバーレスで動くサービスにしたい
- 投げ銭自体もBitCoinやEthereumなどの仮想通貨で決済できるようにして、価値のある投げ銭にしたい
